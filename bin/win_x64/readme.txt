
How to build for windows

 - follow the repo cmake directions
 - run 'cmake --build .' in the root
 - this will build Debug by default
 - try running an example (see examples/Debug)
 - the library is location under src\Debug\glfw3.lib

Release version

 - run: 'cmake --build . --config Release' (in the root dir)
 - see: src\Release\glfw3.lib for the release library
